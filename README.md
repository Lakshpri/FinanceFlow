<h1 align="center">💰 FinanceFlow - Angular Financial Tracker</h1>

<p align="center">
A modern Financial Tracker Application built using Angular (Frontend Only) with LocalStorage.
</p>

<hr>

<h2>🚀 Project Overview</h2>

<p>
FinanceFlow is a complete Angular-based Financial Tracker application developed as part of a structured syllabus-based learning approach.
This project demonstrates core Angular concepts such as components, routing, services, reactive forms, and authentication using localStorage.
</p>

<ul>
  <li>⚡ Fully frontend-based (No backend)</li>
  <li>💾 Uses localStorage for persistence</li>
  <li>🎯 Built strictly based on Angular syllabus</li>
  <li>🧑‍💻 Beginner-friendly and interview-ready</li>
</ul>
