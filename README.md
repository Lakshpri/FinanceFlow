<h1 align="center">FinanceFlow - Financial Tracker Application</h1>

<p align="center">
A modern, responsive Financial Tracker web application built using Angular with localStorage-based persistence and clean architecture.
</p>


<hr>

<h2>📌 Project Overview</h2>

<p>
FinanceFlow is a frontend-only single-page application designed to manage personal finances efficiently. 
It allows users to track income, expenses, and balance with a clean dashboard and structured transaction management system.
</p>

<ul>
  <li>Frontend-only application using Angular</li>
  <li>LocalStorage used for data persistence</li>
  <li>Implements authentication with route protection</li>
  <li>Follows Angular best practices and modular structure</li>
</ul>

<hr>

<h2>✨ Features</h2>

<ul>
  <li>Authentication (Login & Signup)</li>
  <li>Dashboard with financial summary</li>
  <li>Add, Update, Delete Transactions</li>
  <li>Transaction History with filters</li>
  <li>Category-based tracking</li>
  <li>Responsive modern UI</li>
</ul>

<h2>🧠 Angular Concepts Implemented</h2>

<ul>
  <li>Component-based architecture</li>
  <li>Reactive Forms with validation</li>
  <li>Data Binding (property, event, two-way)</li>
  <li>Directives (*ngIf, *ngFor)</li>
  <li>Pipes (currency, date)</li>
  <li>Services & Dependency Injection</li>
  <li>Routing and Navigation</li>
  <li>Auth Guards (CanActivate)</li>
  <li>LocalStorage integration</li>
</ul>
